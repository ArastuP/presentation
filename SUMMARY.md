# Presentation Summary

**Title:** Data management, version control and you
**Author:** Stavrina Dimosthenous
**Date:** 2026-02-19

## Overview

A Reveal.js workshop presentation covering data management and version control for researchers.

## Structure

- `presentation/index.html` — Main slide deck (Reveal.js, solarized theme)
- `presentation/reveal.js/` — Reveal.js framework (submodule)
- `presentation/images/` — Supporting images and GIFs

## Slide Content

1. **Intro** — Title slide
2. **Agenda** — AM: Bash, Git, GitHub remotes; PM: Tidy data, VSCode/Python setup
3. **Gauge** — Mentimeter poll to assess audience familiarity
4. **Unix Shell** — What it is; terminal/console/shell terminology
5. **Command line importance** — Why it matters; FOSS scientific software (e.g. OpenCalphad)
6. **Version control** — What it is and why it matters (file versioning chaos, commented-out LaTeX)
7. **Git** — Is it an important skill?
8. **Module goals** — Learn Unix shell, Git for version control, collaborative GitHub workflows
